// items.txt — Item Registry for System Awakening
//
// Format:
//   *item key "Label" essence_cost
//     Description text. Can span multiple lines.
//     All indented lines become one description string.
//
// Rules:
//   • key must be a single word (letters, numbers, underscores)
//   • essence_cost is the amount of Essence required to purchase (0 is valid for free items)
//   • Purchased items are added to playerState.inventory automatically
//   • description lines must be indented at least one space
//   • a new *item directive or end of file terminates the previous description
